# Projeto Final - Curso HTML e CSS (Origamid)

Projeto desenvolvido como conclusão do curso **HTML e CSS para Iniciantes** da Origamid.

O objetivo foi aplicar conceitos fundamentais de desenvolvimento web, incluindo estruturação semântica, estilização com CSS moderno e criação de formulários funcionais.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- CSS Grid
- Formulários HTML
- JavaScript (validação de formulário)

---

## 📚 Conceitos aplicados

- Estrutura HTML semântica
- Layout responsivo
- Organização de CSS
- Componentização básica
- Validação de formulário com JavaScript
- Boas práticas de desenvolvimento

---

## 📸 Projeto

Este projeto simula um **formulário de solicitação de orçamento**, contendo:

- Campos de dados pessoais
- Seleção de tipo de serviço
- Descrição do projeto
- Validação de formulário
- Feedback visual para o usuário

---

## 🎯 Objetivo do projeto

Consolidar os conhecimentos adquiridos no curso e criar um projeto que possa ser utilizado como **parte do portfólio de desenvolvedor front-end**.
